# Venue Menus Module

Manages venue menus - upload, edit, delete, and preview menu documents (PDF/images).

## Structure

- `ui/` - React components (MenusPage, MenusSection)
- `actions/` - Server actions (menuActions)
- `services/` - Business logic (menusService)
- `schemas/` - Zod schemas for validation (menusSchemas)

## Features

- Upload menu files (PDF/images)
- List all menus for a venue
- Delete menus
- Preview menu files
